# Social-Media-Photo-Resizer
an application that allows users to upload images that they would like to be resized in order to fit the format on multiple platforms, with options given to the user.

#NOTES:

The structure should be:
User triggers upload image from website, then is given an option to generate
on selecting a button like "generate", will trigger the python code
        (could implement an option to choose which social medias the user wants)
Python code with resize all the of the images
Images will get sent to a database
Website pulls from database to show images
Website shows each image with its respective sizing under each app category
